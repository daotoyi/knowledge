## 简述 {#简述}

商标分为一共为 45 类，1到 34 类为商品类别，35 到 45 类为服务类别，每个分类下还有若干服务项。一个商标可以申请多个分类与多个服务项。

## 35 类注册 {#35-类注册}

| 组别 | 说明           |
|----|--------------|
| 3501 | 广告           |
| 3502 | 工商管理辅助业 |
| 3503 | 替他人推销     |
| 3504 | 人事管理辅助业 |
| 3505 | 商业企业迁移   |
| 3506 | 办公事务       |
| 3507 | 财会           |
| 3508 | 单一服务       |
| 3509 | 药品、医疗用品零售或批发服务 |

第 35 类商标包含了广告、销售、服务等广泛行业，哪怕实体行业，也一定要注册第 35 类商标。，尽量多注册几个类目，以免商标被他人抢注。
